# ✈️ SchemaPilot: Precision Function-Calling for Agentic Workflows

**SchemaPilot** is a fine-tuned **Phi-3-mini** model optimized for a single, critical task: converting natural language into 100% valid JSON function calls. 

## 🎯 The Problem
Standard LLMs often "yell" at the user or add conversational filler when they should just be outputting code. **SchemaPilot** is stripped of its personality to maximize its "Agentic Reliability."

## 🚀 Key Features
* **Zero-Talk Inference:** No "Sure, here is your JSON." Just the code.
* **Schema-Strict:** Fine-tuned specifically on your custom API schemas.
* **Ultra-Lightweight:** 3.8B parameters; runs on a 4GB VRAM GPU (or a mobile phone).
